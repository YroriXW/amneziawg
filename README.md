## Unofficial [AmneziaWG](https://amnezia.org) builds
Personal packaging repo for [AmneziaWG](https://github.com/amnezia-vpn/amneziawg-linux-kernel-module) — 
a WireGuard fork with traffic obfuscation. Builds are available for **Debian** and **Fedora** via 
[my OBS repo](https://build.opensuse.org/package/show/home:YroriXW/amneziawg).
